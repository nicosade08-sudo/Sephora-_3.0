## Sephora 3.0 – Redesign Project

A modern redesign of Sephora’s late 1990s website focused on clarity, usability, and visual balance.

### Problem
The original design lacks structure, has heavy visual clutter, and provides little guidance for user navigation. Modern versions can still feel overwhelming due to excessive content and promotions.

### Solution
A cleaner, more structured interface that prioritizes user experience over visual noise.

### Key Improvements
- Clear visual hierarchy
- Simplified navigation
- Reduced visual clutter
- Neutral, modern color palette
- Improved spacing and layout
- Logical content structure (hero, categories, products)

### Tech Stack
HTML, CSS, SCSS

### Note
Includes an experimental bonus page accessible via the “Shop Now” button.
